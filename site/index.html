The three layers

① Survival

Vanilla 1.21.x. No fly, no teleport-to-anywhere, death still hurts. The block game stays sacred.

② Economy

Towny land with rent. Jobs Reborn wages. QuickShop chest shops. Banks with interest. A stock market with player IPOs.

③ Web

This site. Link your MC account, then earn Fisher Pounds via chores, daily check-ins, coaching-hours logged, and minigames.

Why this should work

  • Inflation discipline — taps (wages, web earnings) balanced by drains (rent, fees, cosmetics).
  • Asymmetric roles — miner, farmer, builder, shopkeeper, landlord, financier. The market rewards specialising.
  • Real stakes — miss three days of rent, lose your house. That's the line between a game and a diorama.
  • Low-friction on-ramp — a non-gamer partner can do dishes, earn coins, buy cosmetics, feel part of it.

The plan, at a glance

  1. Phase 0 — Paper server up on fisherserver, playtest with 2.
  2. Phase 1 — CMI + QuickShop + Jobs Reborn + BlueMap. Baseline economy.
  3. Phase 2 — Towny. Land, rent, taxes.
  4. Phase 3 — Auction house, stock market, banks with interest, bounty board.
  5. Phase 4 — Web layer. This site + bridge service + Discord bot.
  6. Phase 5 — Tune, adjust, add money sinks, run events.
